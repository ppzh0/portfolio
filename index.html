Learning Curve (NOW: climbing a mountain)

I started learning about web development on MiMo. It has a pretty much of basics for web development; a good place for starters like me.

It was really difficult transitioning from learning with an app to self-studying. I was too much immersed in gamified-learning system; restarting or creating another account, just for the sake of getting points.

HTML and CSS were easy to understand and use though I learn new things about them every day. When approaching JavaScript, I've tried books and tutorials: Sir Ei Maung's Professional Web Developer 2023, Myanmar Web Developer (YouTube), etc.

But having too much free time and internet distractions, I have done not much of significant advances in learning JavaScript, except knowing what var, let, and basic operators do.
